Installation
============

You can install qmld using pip:

.. code-block:: bash

   pip install qmld

For development installation, clone the repository and install in editable mode:

.. code-block:: bash

   git clone https://github.com/pbowling/qmld.git
   cd qmld
   pip install -e .

Requirements
------------

- Python >= 3.8
- numpy
- sphinx (for building documentation)
